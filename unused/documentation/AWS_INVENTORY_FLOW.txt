QUERY PROCESSING FLOW:
======================

1. User Query Input
   └─► "Show me all RDS instances grouped by engine type"

2. Frontend Processing
   └─► POST /infrastructure-ai with query + context

3. Bot Core API
   └─► Enriches query with AWS resource data
   └─► Calls Infrastructure AI Handler

4. AI Analysis
   └─► Claude AI processes query with RDS context
   └─► Generates insights and recommendations

5. Response Generation
   └─► Structured JSON response with analysis
   └─► Returns to frontend for display

6. UI Update
   └─► Chat interface shows AI response
   └─► Auto-scrolls to new message
